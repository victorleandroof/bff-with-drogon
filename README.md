# bff-with-drogon
BFF with Drogon (C++ web application framework)

## Client Dependencies
```bash
cd client && npm i
```


## BFF Dependencies
[wiki drogon](https://github.com/drogonframework/drogon/wiki/ENG-02-Installation)


## BUILD Client
```bash
cd client && npm run build
```


## Start BFF
```bash
 cd bff
 mkdir build 
 cd build
 cmake ..
 make
 ./bff-with-drogon
```
